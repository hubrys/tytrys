Tytrys - (Tie-Tris)
===================

What is it?
-----------
Tytrys is a tetris clone that runs inside a terminal/console.

What's it written in?
---------------------
Tytrys is written in python and utilizes the curses library to bring the nostalgic blocky goodness to your screen.

Why?
